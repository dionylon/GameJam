# 羊了个羊 Web Clone

这是一个基于 React + TypeScript + TailwindCSS 的《羊了个羊》Web 复刻版。

## 🎮 玩法说明

1.  **目标**：消除所有桌面上的卡牌。
2.  **规则**：
    *   点击卡牌将其移动到下方的槽位中。
    *   槽位中如果有 3 张相同的卡牌，则会消除。
    *   如果槽位满了（7张）且无法消除，游戏失败。
    *   只有最上层（未被遮挡）的卡牌可以点击。

## 🛠 技术栈

*   **React 18**
*   **TypeScript**
*   **Vite**
*   **TailwindCSS**
*   **Zustand** (状态管理)
*   **Lucide React** (图标)

## 🚀 运行项目

1.  安装依赖：
    ```bash
    npm install
    ```

2.  启动开发服务器：
    ```bash
    npm run dev
    ```

3.  构建生产版本：
    ```bash
    npm run build
    ```
